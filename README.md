# poc_delta_table_uv
poc delta lake com uv

### Instalação de UV
~~~cmd
pip install uv
~~~

### Criação de ambiente virtual
~~~cmd
uv venv
~~~

### Activação de ambiente virtual
~~~cmd
# No macOS e Linux.
source .venv/bin/activate
~~~

~~~cmd
# No Windows.
.\.venv\Scripts\activate.ps1
~~~
